# Техническое задание: GoodChoice

## 1. Цель проекта
Разработка Веб-сервиса виртуального гардероба для девушек с функцией подбора готовых стильных образов на каждый день и аналитикой гардероба с визуализацией.

## 2. Роли пользователей
- **Пользователь**: Добавление вещей, создание образов, подбор одежды, просмотр аналитики

## 3. Модели данных
1. **ClothingItem** (Одежда)
   - Название, описание, фото
   - Категория, тип, цвет, сезонность
   - Оценка пользователя, дата добавления

2. **Outfit** (Образ)
   - Название, описание
   - Список вещей (ManyToMany с ClothingItem)
   - Тип мероприятия, погодные условия

3. **WardrobeStats** (Статистика)
   - Статистика использования вещей
   - Графики и аналитика
   - Рекомендации по обновлению гардероба

## 4. Ключевой функционал
* Добавление вещей в виртуальный гардероб с фото
* Подбор одежды по фильтрам (мероприятие, погода, сезон)
* Создание и сохранение готовых образов
* Визуальная аналитика использования гардероба

## 5. Внешние интеграции / Аналитика
* Использование библиотеки Plotly для интерактивных графиков
* Анализ данных с помощью Pandas
